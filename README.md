#  Sarah Homes

Sarah Homes is a modern real estate website designed to showcase properties and allow users to easily connect for enquiries.

##  Features

*  Property listing interface
*  AI chatbot (Sarah AI)
*  WhatsApp integration for quick enquiries
*  Responsive design (mobile + desktop)
*  Fast hosting using Netlify

##  Live Website

https://sarahhomes.netlify.app/

##  Technologies Used

* HTML
* CSS
* JavaScript

## 📌 Key Functionalities

* Users can browse properties
* Chatbot assists with basic queries
* Direct WhatsApp redirection for leads
